How PDF compression works

This tool uses PDF-lib to remove unused objects, compress object streams, and strip unnecessary metadata from your PDF. This is most effective on PDFs that have been edited many times or exported from design software. Scanned image PDFs have limited compression potential since the file size is dominated by image data.
