Why rotate PDF pages?

Scanned documents often come out sideways or upside-down, especially when paper is fed into a scanner incorrectly. PDF Rotator permanently corrects the orientation in the saved file — unlike viewing rotation in a PDF reader which only affects how you see it on screen without saving the change. This tool uses PDF-lib to rotate pages without re-encoding or reducing quality.
