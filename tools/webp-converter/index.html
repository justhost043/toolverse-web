Why use WebP format?

WebP is Google's modern image format that achieves 25–35% smaller file sizes than JPEG at equivalent visual quality, and 25–34% smaller than PNG for lossless compression. Using WebP on your website reduces page load time and improves Core Web Vitals scores. All modern browsers support WebP. The WebP to PNG/JPG direction is useful when you need to edit a WebP file in software that doesn't support it.
