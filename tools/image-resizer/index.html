Which format should I choose?

JPEG is best for photos — it produces small files but uses lossy compression. PNG is best for images with text, logos, or screenshots, and supports transparency, but files are larger than JPEG. WebP is a modern format that typically produces smaller files than both JPEG and PNG at similar quality — it's supported by all modern browsers and is the recommended format for web use.
