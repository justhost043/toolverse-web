How video compression works

Video compression reduces file size by encoding video more efficiently using the H.264 codec. The CRF (Constant Rate Factor) setting controls the quality-size tradeoff — lower values preserve more quality but produce larger files. CRF 23 is the default for H.264 and is a good balance. CRF 28 reduces file size by roughly 50% with a noticeable but acceptable quality reduction for social media sharing.
