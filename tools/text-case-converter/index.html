When to use each case type

UPPERCASE is used for acronyms, emphasis, or headings in some design styles. lowercase is common in brand names and informal writing. Title Case capitalises the first letter of each major word — standard for headlines and book titles. Sentence case only capitalises the first letter of the first word, matching how a normal sentence is written.

camelCase and PascalCase are used in programming — variables and functions in JavaScript often use camelCase, while class names use PascalCase. snake_case is common in Python and database column names. kebab-case is standard for URL slugs and CSS class names.
