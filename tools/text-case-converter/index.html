When to use each case style

UPPERCASE — acronyms, buttons, emphasis, form labels in some design systems. lowercase — brand names (facebook, airbnb), informal writing, email addresses. Title Case — headlines, book titles, article headings. Sentence case — standard for body text and most written content.

camelCase — JavaScript variables, functions (getUserName). PascalCase — class names in OOP languages (UserProfile). snake_case — Python variables, database columns (user_name). CONSTANT_CASE — environment variables, constants (MAX_RETRY_COUNT). kebab-case — URL slugs, CSS classes, HTML attributes. dot.case — configuration keys, package names (com.example.app).
