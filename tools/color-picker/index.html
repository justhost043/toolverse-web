HEX, RGB, HSL and CMYK explained

HEX is the standard for web development — a six-character code like #4F46E5 that compactly represents red, green and blue values in hexadecimal. RGB expresses the same information as three 0–255 numbers (Red, Green, Blue) and is common in CSS and image editing software. HSL (Hue, Saturation, Lightness) is easier for designers to work with intuitively — you can adjust how bright or saturated a color is without changing its base hue. CMYK (Cyan, Magenta, Yellow, Key/Black) is the color model used in print — it describes how much ink of each color is mixed to produce the final color on paper.

What are shades and tints?

A shade is produced by mixing a color with black, making it darker. A tint is produced by mixing with white, making it lighter. The row of swatches shows five shades and five tints of your selected color — useful for building a cohesive design palette where all tones stay harmonious.
