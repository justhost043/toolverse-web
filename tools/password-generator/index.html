How to use the password generator

Pick a length, choose which character sets to include, then click Generate new password. A new random password appears instantly, along with a strength rating. Click Copy to copy it to your clipboard.

Why randomness matters

A password is only as strong as its unpredictability. Patterns based on real words, birthdays, or keyboard sequences (like "qwerty123") can be guessed by automated tools in seconds, no matter how long they are. This generator uses your browser's crypto.getRandomValues() function — the same cryptographically secure source used for encryption — instead of ordinary random number functions that can be predictable.

How to choose password settings

For most accounts, a 12–16 character password mixing all four character types (uppercase, lowercase, numbers, symbols) offers strong protection against brute-force attacks. For highly sensitive accounts — email, banking, or your password manager's master password — consider going up to 20+ characters. Avoid disabling symbols and numbers purely for convenience; every character set you remove makes the password meaningfully easier to crack.

What this tool does not do

This generator does not store, log, or transmit any password it creates — generation happens entirely on your device. It also will not remember your password for you; once you navigate away, it's gone unless you've saved it somewhere secure, such as a dedicated password manager.

Frequently asked questions

Are the generated passwords stored anywhere?

No. Passwords are generated locally in your browser using the Web Crypto API and are never sent to or stored on any server.

How long should my password be?

Most security guidelines recommend at least 12 characters, mixing uppercase, lowercase, numbers and symbols.

Is this generator cryptographically secure?

Yes. It uses the browser's built-in Web Crypto API (crypto.getRandomValues), the same source used for cryptographic operations, instead of Math.random().
