What makes a strong password?

A strong password has three properties: length (longer is dramatically stronger — each character multiplies entropy), character variety (mixing uppercase, lowercase, numbers, and symbols), and true randomness (this tool uses crypto.getRandomValues() — the same cryptographic random source used in SSL/TLS). A 20-character random password with mixed characters has approximately 130 bits of entropy — computationally infeasible to brute-force.

Are passphrases more secure than passwords?

Passphrases like "Correct-Horse-Battery-Staple" are often both easier to remember and cryptographically strong. Four common words gives roughly 51 bits of entropy — reasonable for most accounts. Five or six words with numbers and separators approaches or exceeds the strength of a random character password, while being much more memorable.

Is this password generator safe?

Yes. Passwords are generated entirely in your browser using the Web Cryptography API. No password data is transmitted over the internet, stored on a server, or logged anywhere. The source code is auditable in your browser's developer tools.
