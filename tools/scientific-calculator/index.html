Keyboard shortcuts

This calculator supports full keyboard input. Numbers 0–9 and operators (+, −, *, /) work directly from the keyboard. Press Enter or = to calculate. Backspace removes the last character. Escape clears the input. Parentheses ( and ) work as expected for grouping operations.
