Why minify CSS?

CSS files are downloaded every time a page loads. Minifying removes every unnecessary character — spaces, newlines, comments, redundant semicolons — without changing the visual result. A stylesheet that's 40% smaller loads and parses that much faster, contributing to better Core Web Vitals scores and a faster perceived page load, especially on mobile connections.
