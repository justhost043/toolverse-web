Markdown table syntax

Markdown tables use pipe characters (|) to separate columns and hyphens (---) to create the header separator row. Column alignment is controlled by colons in the separator: :--- for left, :---: for center, and ---: for right alignment. This generator handles all the formatting automatically — just fill in your data.
