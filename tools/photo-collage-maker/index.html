Create photo collages for social media

Photo collages are a popular format for Instagram, Pinterest, and blog posts. A 2x2 grid works well for product photos or before/after comparisons. The 2-column layout is classic for portrait photography. The 1-large + 2-small layout (5-photo) creates visual hierarchy that draws attention to a hero image while supporting it with context photos.
