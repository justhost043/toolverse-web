How does text flipping work?

Most of these transformations use Unicode characters that visually resemble flipped or stylized versions of Latin letters. For example, the upside-down ╔Р is a real Unicode character (U+0250) that looks like a flipped 'a'. By substituting each letter with its Unicode equivalent, we can create text that appears flipped or mirrored while still being copy-pasteable as plain text.

The Bold and Italic styles use Unicode Mathematical Alphanumeric Symbols тАФ these are real Unicode characters designed for mathematical notation that happen to look like bold or italic Latin letters when used in regular text. This is why bold text in Instagram bios or Twitter posts works without any special formatting.
