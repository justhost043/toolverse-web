What is Base64 encoding?

Base64 converts binary data into a string of 64 printable ASCII characters. It is used wherever binary data needs to be transmitted or stored in a text-only format — HTTP headers, JSON APIs, CSS data URIs, HTML embedded images, and email attachments.

When to use URL-safe Base64

Standard Base64 uses the characters + and /, which have special meaning in URLs. URL-safe Base64 replaces these with - and _, making the encoded string safe to include in URLs, query parameters, and JWT tokens without percent-encoding.

How to embed an image in HTML/CSS

Use the File → Base64 tab to encode an image. Copy the output and use it directly in HTML as src="data:image/png;base64,..." or in CSS as background-image: url("data:image/png;base64,..."). This eliminates an HTTP request for the image but increases file size by ~33%.
