What is Base64 encoding?

Base64 is an encoding scheme that converts binary data into a string of ASCII characters. It is used to safely transmit binary data (like images or files) over systems that only support text — for example, embedding an image directly in HTML or CSS using a data URL, storing binary data in JSON, or sending files in email attachments. Base64 is not encryption — it is purely an encoding format and can be decoded by anyone.

URL-safe Base64

Standard Base64 uses + and / characters which have special meanings in URLs. URL-safe Base64 replaces + with - and / with _, making it safe to use in URLs and file names without percent-encoding.
