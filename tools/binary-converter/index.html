Number systems explained

Decimal (base 10) is the everyday number system — digits 0–9. Binary (base 2) is the language of computers — everything is represented as 0s and 1s. Hexadecimal (base 16) uses digits 0–9 and letters A–F; it's commonly used in programming, color codes (#FF5733), and memory addresses because one hex digit represents exactly 4 binary bits. Octal (base 8) uses digits 0–7 and was historically used in computing; it's still used in Unix file permissions (chmod 755).
