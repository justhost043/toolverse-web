SVG conversion and optimization

SVG (Scalable Vector Graphics) is an XML-based format that scales to any size without quality loss — ideal for logos, icons, and illustrations. Converting SVG to PNG or JPG creates a fixed-resolution raster image for use in contexts that don't support SVG. SVG optimization removes unnecessary metadata, editor comments, and whitespace from SVG files exported by tools like Illustrator or Figma, reducing file size by 30-70% while preserving visual fidelity.
