    Is it safe to type my password here?

    Yes. This checker runs entirely in your browser using JavaScript. Your password is never transmitted over the network or stored anywhere. You can verify this by disconnecting from the internet and reloading the page — the checker works perfectly offline, which is proof that nothing leaves your device.

    What is password entropy?

    Entropy measures unpredictability in bits. Each bit doubles the number of possible passwords — 40 bits means about 1 trillion possibilities, 60 bits means about 1 quintillion. The higher the entropy, the more time it takes a computer to crack the password by trying every combination.
