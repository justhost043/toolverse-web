Is this truly random?

This generator uses your browser's crypto.getRandomValues() — the same cryptographically secure source used in the password generator. It's far more random than Math.random(), which uses a predictable algorithm. For lotteries, games, or any purpose where fairness matters, cryptographic randomness is the right choice.

Common uses for a random number generator

Picking a lottery number, deciding who goes first in a game, selecting a random item from a list, running a classroom raffle, making a decision when you're stuck between options, rolling dice for board games or RPGs online, or generating sample data for testing software.
