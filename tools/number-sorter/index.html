When to use a list sorter

List sorters are useful whenever you need to organize data quickly without opening a spreadsheet. Common uses include sorting a list of names alphabetically for a class register, ordering product prices numerically to find the highest or lowest, randomizing a list of names for a raffle draw, removing duplicate email addresses from a collected list, or reversing the order of a numbered list to put the newest items first.
