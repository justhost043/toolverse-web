What are HTML entities?

HTML entities are special codes used to represent characters that have reserved meanings in HTML or that can't be typed directly. The five characters with special HTML meaning are < (less-than, start of a tag), > (greater-than, end of a tag), & (ampersand, start of an entity), " (double quote, used in attributes), and ' (single quote). These must be encoded as &lt;, &gt;, &amp;, &quot;, and &apos; respectively when appearing as literal text content in HTML.
