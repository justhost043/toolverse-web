About this stopwatch

This stopwatch runs at millisecond precision using the browser's performance.now() API, which is more accurate than Date.now() for timing purposes. Lap times are recorded instantly and highlighted in green (fastest) and red (slowest) so you can compare splits at a glance. All data stays in the browser — nothing is sent anywhere.
