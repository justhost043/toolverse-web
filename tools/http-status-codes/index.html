HTTP status code categories

1xx (Informational) indicates the request was received and processing continues. 2xx (Success) means the request was successfully received and processed. 3xx (Redirection) means further action is needed to complete the request, usually a redirect to a different URL. 4xx (Client Error) indicates the request contains bad syntax or cannot be fulfilled due to the client's fault. 5xx (Server Error) indicates the server failed to fulfill a valid request.
