How to scan a QR code with this tool

Click "Start Camera" to use your device camera — point it at any QR code and it will be decoded automatically within a second or two. Alternatively, switch to "Upload Image" and drop or select any image file containing a QR code. The decoder works entirely in your browser using the jsQR library — no image is uploaded to any server.

What types of QR codes can this scan?

This scanner decodes standard QR codes including URL links, plain text, Wi-Fi network credentials (WIFI: format), contact cards (vCard format), email addresses, phone numbers, SMS messages, and any other content that can be stored in a standard QR code. It uses the jsQR open-source library for decoding.
