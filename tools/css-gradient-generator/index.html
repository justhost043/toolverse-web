CSS gradient types explained

Linear gradients transition colors along a straight line at any angle. They're the most commonly used gradient type in web design — for backgrounds, buttons, headers, and dividers. The angle determines the direction: 0° goes bottom to top, 90° goes left to right, 135° goes diagonally.

Radial gradients radiate outward from a center point in a circular or elliptical shape. They create a spotlight or glow effect, useful for hero backgrounds, card hover states, and decorative elements.

Conic gradients rotate colors around a center point — like a pie chart or color wheel. They're newer but well-supported in modern browsers and useful for creating pie charts, color wheels, and distinctive backgrounds purely in CSS with no images needed.

How many color stops can I add?

CSS gradients support unlimited color stops. Most practical designs use 2–4 stops. Adding more stops gives you finer control over exactly where each color begins and ends — useful for sunset-style multi-color gradients. Each stop has a percentage position (0% = start, 100% = end) that you can fine-tune here.
