What is a text diff?

A diff (from "difference") shows exactly what changed between two versions of text. Added lines are highlighted in green, removed lines in red, and unchanged lines are shown in a neutral color. Developers use diffs to review code changes; writers use them to compare document drafts; editors use them to track revisions.

Common uses for a diff tool

Comparing two versions of a contract or legal document to spot what changed between drafts. Reviewing changes made to a configuration file or code snippet. Checking whether a translated text matches the original structure. Verifying that a copied article hasn't been altered. Tracking changes between two versions of a blog post or marketing copy.
