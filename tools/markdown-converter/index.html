What is Markdown?

Markdown is a lightweight text formatting syntax created by John Gruber in 2004. The core idea is that plain text should be readable as-is, but also convertible to rich HTML. Writers use it for README files, documentation, blog posts, and any content that will eventually be published as HTML but needs to be authored in a readable plain-text format.

Markdown cheat sheet

Use # Heading 1, ## Heading 2 for headings. Wrap text in **bold** or *italic*. Create links with [text](url). Add inline code with backticks and code blocks with triple backticks. Prefix lines with > for blockquotes, - or * for bullet lists, and numbers for ordered lists.
