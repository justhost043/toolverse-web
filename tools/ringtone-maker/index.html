How to set a custom ringtone

On iPhone: download the WAV file, rename it to .m4r (you can do this on a PC), then drag it to iTunes and sync. On Android: download the file, move it to the Ringtones folder using a file manager, then go to Settings > Sounds > Phone Ringtone and select it. Maximum ringtone length is typically 30-40 seconds on most phones.
