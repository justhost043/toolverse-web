Audio format conversion explained

Different audio formats use different compression algorithms. OGG Vorbis, AAC, and WebM are compressed formats that reduce file size by removing audio data the human ear theoretically cannot detect. WAV stores raw uncompressed audio. When you convert a compressed file to WAV, the file becomes larger but no further quality is lost — you are simply changing the container, not re-compressing the audio.
