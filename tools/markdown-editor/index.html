Markdown syntax quick reference

Markdown is a lightweight markup language that converts to formatted HTML. Use # H1, ## H2, ### H3 for headings. **bold** for bold, *italic* for italic. Backticks for inline code and triple backticks for code blocks. Hyphens for bullet lists, 1. for numbered lists. [text](url) for links. Markdown is used in README files, GitHub, GitLab, Notion, Obsidian, and most documentation platforms.

GitHub Flavored Markdown (GFM)

This editor supports GFM — the Markdown superset used on GitHub. It adds: fenced code blocks with ``` language hints, tables with pipe syntax, ~~strikethrough~~ with tildes, task lists with - [ ], and autolinked URLs. GFM is also supported by GitLab, Bitbucket, Jira, and most developer tools.
