Markdown syntax quick reference

Markdown is a lightweight markup language that converts plain text to formatted HTML. # Heading creates an H1, **bold** makes text bold, *italic* makes it italic, and backticks create inline code. Hyphens create bullet lists, and you can create links with [text](URL). Triple backticks create code blocks. Markdown is widely used in README files, blogs, documentation, and messaging apps like Slack and Discord.
