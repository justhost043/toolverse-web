What JSON structure is supported?

This converter works with a JSON array where each element is an object — the most common format returned by REST APIs. The keys of the first object become the CSV column headers, and each object becomes a row. If the "Flatten nested objects" option is enabled, nested objects are flattened with dot notation (e.g., address.city), making them accessible as separate columns.
