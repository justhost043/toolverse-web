What does this JSON formatter do?

Paste raw or minified JSON to instantly format it with proper indentation and syntax highlighting. The validator catches errors in real time — mismatched brackets, trailing commas (not allowed in JSON), missing quotes on keys, and unescaped characters. Sort Keys rearranges all object properties alphabetically, making it easier to compare two JSON objects or find a specific key in a large payload.

Format vs Minify

Formatting adds indentation and line breaks for human readability — essential when debugging API responses or config files. Minifying removes all whitespace, reducing file size — use this before including JSON in a production build or sending over a network where payload size matters.

Common JSON errors

The most frequent JSON mistakes are: trailing commas after the last array or object item (valid in JavaScript but not in JSON), missing double quotes around property names (JSON requires double quotes — single quotes are not valid), and unescaped backslashes or control characters inside strings.
