How the encryption works

This tool uses the browser's built-in Web Crypto API with AES-GCM 256-bit encryption. Your password is never stored anywhere — it's used to derive an encryption key using PBKDF2 (a standard key derivation function) each time you unlock. The encrypted notes are stored in your browser's localStorage. Without the correct password, the encrypted data is completely unreadable. Even if someone accessed your browser storage, they would see only random bytes.

Important limitations

Because notes are stored in localStorage, they are tied to this browser on this device. Clearing your browser data will delete your notes. If you forget your password, there is no recovery — the encryption is genuine, not a simulation. For notes you want to keep long-term, consider copying the text to a backup location.
