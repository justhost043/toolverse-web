What HTML tags are converted?

This converter handles the most common HTML elements: headings (h1–h6), paragraphs, bold and italic text, links, images, unordered and ordered lists, blockquotes, code and pre blocks, horizontal rules, and basic tables. It strips unknown or complex tags while keeping their text content.

Common use cases

Developers converting CMS-exported HTML content to Markdown for use in static site generators like Jekyll, Hugo or Next.js. Writers migrating blog content from WordPress or other HTML-based platforms to Markdown-first tools. Anyone who receives HTML emails or documents and wants to paste the content into a Markdown editor.
