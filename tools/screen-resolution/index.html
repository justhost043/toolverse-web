Screen resolution vs viewport size

Your screen resolution is the total number of physical pixels on your display — for example, 1920×1080 (Full HD). Your viewport is the area of the browser window that's actually showing web content — this is smaller than the screen resolution because it excludes the browser's toolbar, address bar, bookmarks bar, and window borders. When you resize the browser window, the viewport changes but your screen resolution stays the same.

What is device pixel ratio (DPR)?

DPR is the ratio between physical pixels and logical pixels. On a standard screen, DPR is 1 — one CSS pixel equals one physical pixel. On Retina and high-DPI screens (common on modern MacBooks, iPhones and Android flagships), DPR is 2 or more, meaning 4 physical pixels are used to render each CSS pixel, resulting in sharper text and images. A DPR of 3 is common on high-end phones like recent iPhones. Web developers use DPR to serve appropriate image resolutions for each screen type.
