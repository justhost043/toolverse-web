What is a UUID?

A UUID (Universally Unique Identifier) is a 128-bit label used to identify information in computer systems. Written as 32 hexadecimal characters in a specific pattern (xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx), UUIDs are designed to be unique across all computers and all time without requiring a central registration authority. They're used as primary keys in databases, file names, session tokens, and anywhere a guaranteed-unique identifier is needed.

UUID v4 vs v1

UUID v4 is generated using random or pseudo-random numbers. It has no predictable structure and is the most widely used version for general purposes — it's what you should use when you just need a unique ID. UUID v1 is generated from the current timestamp and the MAC address of the generating machine, making it time-sortable. This is useful when you need to order records by when they were created, but it exposes the generating machine's network address.
