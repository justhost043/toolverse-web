How to merge PDF files

Click the upload area or drag your PDF files directly onto it — you can select multiple files at once. The files appear in a list below; drag them up or down to set the order they'll appear in the final merged document. When you're happy with the order, click Merge PDFs and download the combined file.

Is this PDF merger safe to use?

Yes. This tool uses pdf-lib, an open-source JavaScript library that runs entirely inside your browser. Your PDF files are never sent to a server — all merging happens locally on your device, making it safe for confidential documents, contracts, medical records, or any sensitive file.

File size and page limits

Because everything runs in your browser, the practical limit depends on your device's RAM rather than any server restriction. Most modern computers can comfortably merge PDFs totalling 50–100MB. For very large files (hundreds of MB), a desktop PDF application may be faster.
