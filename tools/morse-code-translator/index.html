What is Morse code?

Morse code is a communication system that represents letters and numbers as sequences of dots (short signals) and dashes (long signals). It was invented in the 1830s by Samuel Morse and Alfred Vail for use with the electric telegraph. Before telephone and radio, it was the primary method of long-distance communication — and it's still used today by amateur radio operators worldwide.

How to read Morse code

Each character is made up of dots (·) and dashes (−). A dot is a short signal; a dash is three times as long as a dot. Characters are separated by a short pause, and words are separated by a longer pause. The most famous Morse code sequence is SOS (· · · − − − · · ·) — three dots, three dashes, three dots — used as an international distress signal.
