CSS box shadow syntax

The CSS box-shadow property takes these values in order: horizontal offset, vertical offset, blur radius, spread radius, and color. A positive horizontal offset moves the shadow right; negative moves it left. A positive vertical offset moves it down; negative moves it up. The blur radius controls how soft or sharp the shadow is — 0 gives a hard-edged shadow. The spread radius expands or contracts the shadow's overall size. Adding the inset keyword places the shadow inside the element instead of outside.

Layering multiple shadows

CSS allows multiple box shadows on a single element by comma-separating them. This enables complex effects like combining a soft ambient shadow with a sharper contact shadow below an element — a technique used in Material Design. The shadows are rendered from the first in the list (on top) to the last (on bottom).
