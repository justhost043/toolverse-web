Speed change vs pitch change

When you change audio speed using this tool, the playback rate changes but the Web Audio API naturally preserves pitch at moderate speed changes (up to 2x). For extreme values like 4x, some pitch shift is expected. This is ideal for speeding up recordings, podcasts, or lectures while keeping the voice recognizable.
